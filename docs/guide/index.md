# Lume UI 介绍

Lume UI 是一个现代化的 React 组件库，基于 Radix UI 和 Tailwind CSS 构建。

## 🎯 设计理念

### 可访问性优先

基于 Radix UI，确保所有组件符合 WAI-ARIA 标准，提供优秀的无障碍体验。

### 灵活定制

使用 Tailwind CSS，支持高度自定义，轻松适配各种设计系统。

### TypeScript 支持

完整的类型定义，提供最佳的开发体验。

### 按需引入

支持 Tree-shaking，只打包使用的组件。

## 📦 特性

- 🎨 **现代化设计** - 简洁美观的默认样式
- ♿ **可访问性** - 基于 Radix UI，符合 ARIA 标准
- 🎯 **TypeScript** - 完整的类型支持
- 🌲 **Tree-shaking** - 按需加载，优化包体积
- 🎭 **主题定制** - 基于 Tailwind CSS，灵活定制
- 📦 **Monorepo** - 清晰的代码组织
